Getting Started

You need River 0.4+ first, then Triad on top. The installer handles the login session entry.

  • Install on Void, Arch, Debian, or NixOS
  • Launch from your display manager or a TTY
  • Validate with triad msg state
{% set cfg = get_section(path="configuration/_index.md") %}

Configuration

Your config lives at ~/.config/triad/config.kdl and reloads on save — no restart needed.

  • Set key bindings, startup apps, and shell/bar
  • Write window rules to control placement and floating
  • Configure monitors, workspaces, and layouts
{% set use = get_section(path="usage/_index.md") %}

Usage & IPC

Control Triad from the keyboard or any script that can open a Unix socket.

  • Switch layouts mid-session with a binding or triad msg
  • Move windows across workspaces and monitors
  • Automate placement with Janet scripts
