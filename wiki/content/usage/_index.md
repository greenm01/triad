# Usage

Control Triad through IPC commands (`triad msg`), key bindings, and Janet scripts.

Explore commands in [IPC & Commands](@/usage/ipc-commands.md) and scripting in the dedicated Janet section.

Tags and rules drive placement. Scripts react to events for dynamic behavior.
