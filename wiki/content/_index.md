# Welcome to the Triad Wiki

Triad is a programmable Wayland window manager built for River. River manages the protocol; Triad owns placement, policy, IPC, and scripting.

Triad keeps your session as data. Windows wear stable tags. Rules and scripts decide placement from live state. Restart Triad and nothing moves.
