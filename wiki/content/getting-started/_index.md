# Getting Started

Install River first, then Triad. Launch a River (Triad) session from your display manager or TTY. The first run creates a starter config at `~/.config/triad/config.kdl`.

See the [installation guide](@/getting-started/installation.md) for full steps on Linux distributions, NixOS, and testing methods.

After launch, validate your setup with `triad validate-config` and `triad msg state`.
